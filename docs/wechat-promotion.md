```markdown
# 我给 Mac 装了个「AI 管家」，从此动嘴就能干活

## 痛点

每天用 Mac 工作，你是不是也这样：

- 调亮度？去系统设置翻半天
- 搜文件？打开 Finder 慢慢找
- 翻译？打开浏览器、打开谷歌翻译、复制粘贴
- 连 VPN？打开终端、敲命令、输密码
- 发通知？打开企业微信、找群、打字、发送

一个简单的操作，要经过 N 步才能完成。

## 解决方案

Mac Agent Personal —— 一个开源的 Mac 桌面 AI 助手。

装上之后，你只需要「说」就行了：

```
你：太亮了
Mac：好的，亮度已调到 40%

你：帮我搜一下合同
Mac：找到 5 份合同文件

你：翻译这一段
Mac：已翻译为英文

你：连一下 VPN
Mac：VPN 已连接

你：通知群里下午开会
Mac：已在企业微信群发送通知
```

文件管理：搜索、压缩、解压、图片转换、重复文件检测

## 为什么选择它？

### 1. 中文原生支持

不需要记任何英文命令。
说「亮度」就是调亮度，说「温度」就是查 CPU 温度。
157 条中文别名，你说什么它都懂。

### 2. 本地运行，完全离线

所有命令都在本地执行，不需要联网。
你的数据不会离开你的电脑。

### 3. 开源免费

MIT 协议，源码全公开。
8088 行 Python 代码，每一行都看得见。

### 4. 支持 AI 对话

可选接入 Ollama 本地大模型或 OpenAI，
让它真正「理解」你的意图。

```
你：我电脑好像有点卡
→ 它自动查 CPU、内存、磁盘
→ 告诉你哪里出了问题
→ 给出优化建议
```

## 快速上手

前提：macOS 11.0+，已安装 Python 3.10+

```bash
# 安装
brew install ollama
ollama serve
ollama pull qwen3:8b

pip install knowagent-personal

# 开玩
ka
```

然后直接说你想做什么就行。
