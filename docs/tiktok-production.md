# 抖音短视频制作指南

时长：30-45 秒  
工具：CapCut（剪映，免费）  
音乐：轻快卡点（剪映搜索"科技感"、"极简"）

---

## 第一步：屏幕录制

用 macOS 自带的 QuickTime Player 录屏：

1. 打开 QuickTime Player → 文件 → 新建屏幕录制
2. 选择录制区域（全屏或窗口）
3. 确保 **Terminal 字体调大**（Cmd+Plus 放大）
4. 按以下脚本逐段录制

---

## 第二步：逐段录制脚本

### 片段 1 — 开场 (0-5s)

```
画面：Mac 桌面干净状态
操作：打开 Terminal，输入 ka 并回车
字幕：🧠 Mac Agent Personal
音效：键盘打字声
```

### 片段 2 — 系统状态 (5-10s)

```
画面：ka REPL 界面
操作：输入"系统状态"回车
效果：显示 CPU/内存/磁盘信息
字幕：说句话就能看系统状态
```

### 片段 3 — 播放音乐 (10-17s)

```
画面：切到新的 Terminal 窗口
操作：输入"ka 播放周杰伦的歌"
效果：Apple Music 弹出 + 播放预览
字幕：🎵 动口不动手
```

### 片段 4 — 截图 OCR (17-24s)

```
画面：切到新的 Terminal 窗口
操作：输入"ka 看看屏幕上有什么字"
效果：截屏 → OCR 识别出文字
字幕：📸 截屏自动识别文字
```

### 片段 5 — RAG 知识库 (24-30s)

```
画面：切回 Terminal
操作：输入"ka rag search 机器学习"
效果：显示文档搜索结果
字幕：📚 搜自己的笔记
```

### 片段 6 — 安装方式 (30-38s)

```
画面：显示 GitHub 页面（浏览器）
字幕：👇 开源免费
       pip install knowagent-personal
       GitHub 搜 knowagent-personal
```

### 片段 7 — 结尾 (38-45s)

```
画面：黑色背景 + 大字
文字：关注我 ❤️ 更多 Mac 效率工具
```

---

## 第三步：剪映（CapCut）剪辑

### 1. 导入素材
- 把录好的 6 段视频导入剪映
- 按上面的顺序排列

### 2. 添加文字
- 每段视频顶部加大字标题
- 字体：**思源黑体 Bold**
- 颜色：白色 + 黑色描边
- 大小：覆盖屏幕 1/3

### 3. 添加贴纸
- 每段视频对应的 emoji：🎵 📸 📚 📧
- 放在画面右下角

### 4. 配乐
- 剪映搜索 "Upbeat Technology" 或 "科技感"
- 音量调低至 20%（背景音不要盖过阅读体验）

### 5. 导出设置
- 分辨率：1080×1920（竖屏）
- 帧率：30fps
- 格式：MP4
