## 描述

### 第一段（简介）

> Mac Agent Personal is an open-source, local-first AI assistant for macOS. It can control Apple Music, read/send emails, take screenshots with OCR, automate UI interactions via Accessibility API, search personal documents with local RAG, and more — all running 100% offline with no data leaving your machine.

### 第二段（核心能力）

> Unlike cloud-based assistants that can't interact with your desktop, Mac Agent Personal is deeply integrated into macOS. It uses native Swift tools for UI automation (Accessibility API) and OCR (Vision framework), direct AppleScript integration for Mail and Music, and local LLM (Ollama) for natural language understanding — all without sending your data to any server.

### 第三段（为什么不同）

> While Copilot and ChatGPT Desktop are great for conversation, they can't control your system. Mac Agent Personal fills this gap: it's an agent that actually *does things* on your Mac. And it's fully open source (MIT), so you can extend it however you want.
